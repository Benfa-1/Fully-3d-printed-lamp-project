# Fully-3d-printed-lamp-project

I worked with two separate CAD models and modified them so they fit together as one lamp design. I resized and adjusted the base to match the lampshade and fit the hardware I was using. I needed a smaller lamp with a specific look and a specific function and instead of settling for what was available, I altered and designed the parts to get the size, fit, and style I wanted into one final design.

Printing the lampshade in vase mode was way harder than I thought. It took a lot of tweaking to get it to print clean without messing up, and it taught me a lot about how my printer behaves, especially with thin walls and long continuous prints.

The base was a lot of work too. I had to change pretty much everything so it would actually fit my hardware, because no one really makes a lamp this small that still uses a standard bulb socket. I had to line everything up super carefully so the bulb sits correctly and nothing makes contact or melts.

I ended up using LED bulbs because the clearance was just too tight for a normal bulb. A regular one would’ve gotten too hot and probably melted parts of the lamp. That restraint made me think a lot about the design so it ended up small, safe, and still working the way I wanted.
